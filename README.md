# Engo (잉고)

공부한 영어 표현·문장·문법을 직접 정리하는 윈도우 프로그램입니다.
정리한 것은 바탕화면에 띄우는 복습 메모지로 다시 확인할 수 있습니다.
서버 없이 이 컴퓨터에만 저장하고, 다른 기기와는 파일을 주고받아 합칩니다.

## 실행

**`Engo.bat`** 을 실행하면 됩니다.

Python 이 없으면 어디서 받는지 알려주고, 있으면 알아서 찾아 씁니다
(`py` 런처 → `PATH` → 표준 설치 위치 순). 처음 한 번은 무엇을 내려받을지
확인 창이 뜨고, 그 다음부터는 바로 시작합니다.

## 탭

### 1. 영어 표현

공부한 표현을 원문·한글 뜻·메모·출처·태그로 정리합니다.
왼쪽 표에서 항목을 고르면 오른쪽에서 고칠 수 있고, 새로 쓰면 새 항목이 됩니다.
`Ctrl+S` 로 저장하며, 저장하면 바로 다음 항목을 쓸 수 있는 상태가 됩니다.
위쪽 검색창과 태그로 걸러 볼 수 있습니다.

### 2. 원문 해석

영어 원문을 붙여넣으면 **문장마다 한 행**으로 나뉩니다.
행마다 `내 해석` 칸에 직접 해석을 적고, `피드백 메모` 칸에 스스로 남길 말을 적습니다.
왼쪽 목록에서 지문을 고르거나 새로 추가합니다.

### 3. 외우고 싶은 문장

특히 외우고 싶은 문장을 원문·번역으로 정리합니다.
⭐ 를 켜면 목록 위쪽에 모입니다.

### 4. 문법

`주요 표현` 칸과 `설명` 칸을 나눠서 정리합니다. 예문 칸도 있습니다.

### 5. 데이터

내보내기 / 가져오기(합치기) / CSV / 백업, 그리고 프로그램 업데이트와 읽어주기 음성 설정.

다른 기기로 옮길 때는 `파일로 내보내기` → 그 파일을 옮긴 뒤 상대 기기에서 `합치기`.
같은 항목이 양쪽에 있으면 **더 나중에 수정한 쪽**이 남고, 같은 파일을 두 번 합쳐도 결과는 같습니다.

맨 아래 `프로그램 삭제` 는 Engo가 만든 데이터와 내려받은 구성 요소만 지웁니다.

## 복습 메모지

트레이 아이콘 우클릭 → `📌 복습 메모지`, 또는 메인 창에서 `Ctrl+N`.

창틀 없는 메모지가 뜨고, 제목 줄을 끌어 옮깁니다.
기본은 **한글 뜻을 가린 상태** — 항목을 클릭하면 뜻이 열리고 `알아요` / `헷갈려요` 를 고릅니다.
`알아요` 는 복습 단계를 올리고 목록에서 빼며, `헷갈려요` 는 처음 단계로 되돌립니다.
위치·크기·색은 기억되어, 프로그램을 껐다 켜도 띄워뒀던 메모지가 그대로 돌아옵니다.

## 메모리

| | |
|---|---|
| 트레이에만 있을 때 (창 닫음) | 약 47 MB |
| 메인 창을 열고 5개 탭을 모두 본 뒤 | 약 150 MB |
| 복습 메모지 1장당 | 약 6 MB |

메인 창을 닫으면 창에 딸린 위젯 376개가 **전부 해제**되고 트레이 아이콘만 남습니다.
다시 열 때 새로 만듭니다. 표는 한 번에 200행씩만 읽어, 항목이 수천 건이어도 사용량이 늘지 않습니다.

## 저장 위치

```
%APPDATA%\Engo\
    study.db          학습 데이터 전부 (표현·문장·문법·지문·설정·메모지 위치)
    study.db-wal      쓰기 로그
    study.db-shm      공유 메모리
    backups\          '지금 백업' 으로 만든 사본
    speech*.wav       읽어주기 임시 파일
```

보통 `C:\Users\<사용자>\AppData\Roaming\Engo\` 입니다.
`데이터` 탭 위쪽에 실제 경로가 표시됩니다.

읽어주기 음성 파일은 프로그램 폴더 안 `voices\` 에 들어갑니다 (기본 4종, 약 336MB).
